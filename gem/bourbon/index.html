Gems

Bourbon, Neat, & Bitters

Installation

To install all three type in..

$ gem install bourbon
$ gem install neat
$ gem install bitters

Then, to install them into your styles directory, type..

$ bourbon install
$ neat install
$ bitters install

for more information check out this gem's home page.
