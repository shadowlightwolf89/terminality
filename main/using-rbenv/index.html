Using rbenv

Installing Ruby Versions

rbenv install -l

To list all available versions,

then

rbenv install 2.2.2

for example, to install ruby version 2.2.2, if you want to then, uninstall your ruby version..

rbenv uninstall 2.2.2

If you want a single, globally used version to always be accessible when you first start the terminal, add this to your .bash_profile or .zshrc

export RBENV_VERSION=2.2.2
