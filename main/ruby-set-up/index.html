Ruby Set Up

Installation

This post explains how to install Ruby Gems on Windows or Linux systems, I don't have instructions for Mac, as I do not own one.

Ubuntu

There are two methods to installing ruby, the first, is to simply install it on it's own, like so..

$ apt-get install ruby ruby-dev

The second method is to install rbenv and ruby-build, which will allow you to switch ruby versions on the fly, installation is done like this..

$ git clone https://github.com/sstephenson/rbenv.git ~/.rbenv
$ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile

or, if you use zsh

$ git clone https://github.com/sstephenson/rbenv.git ~/.rbenv
$ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.zshrc
$ echo 'eval "$(rbenv init -)"' >> ~/.zshrc

Then restart the shell, and type

$ type rbenv

to see if it has installed properly.

In order to get ruby-build on Linux, you should type the following..

$ git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build

Windows

To install ruby on windows, download the Ruby for Windows Binary and install it, you'll also need the devkit from the same site.

It is recommended to extract the devkit to C:\RubyDevKit once that's done, type the following into a terminal opened in the extracted folder..

$ ruby dk.rb init
$ ruby dk.rb install
