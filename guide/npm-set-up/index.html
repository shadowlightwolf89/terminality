NPM Set Up

Installation

This post explains how to install NPM on Windows or Linux systems, I don't have instructions for Mac, as I do not own one.

Ubuntu

There are two and a half methods to installing NPM, the first two are to install either NodeJS or iojs manually, the second is to install nvm, which is done like so..

$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.25.4/install.sh | bash

or, if you use zsh

$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.25.4/install.sh | bash

Then restart your shell

Windows

To install NPM on Windows, download either NodeJS or iojs and install it.
